# Garage Manager

> Учебный проект на C#/.NET: консольное приложение для учёта обслуживания автомобиля и расходов.

Garage Manager — это практичный учебный проект, созданный для решения реальной жизненной задачи: ведения журнала работ по машине и связанных расходов. Проект помогает прокачать базовые навыки backend-разработки: структуры данных, ООП, работу с файлами (JSON), архитектуру и чистую структуру проекта.

## Фон

Проект рассчитан на постепенную эволюцию: от простой консольной версии к хранению данных в JSON, затем к поиску и фильтрации, далее к подключению базы данных и, в перспективе, к веб-версии на ASP.NET Core.

## Установка

1. Установите .NET SDK (версию, совместимую с проектом).
2. Клонируйте репозиторий:

```bash
git clone <repo-url>
cd GarageManager
```

## Использование

Запустите консольное приложение из решения:

```bash
dotnet run --project GarageManager
```

## Развитие проекта

Текущая цель — пройти эволюционные этапы:

1. Консольный интерфейс и работа в памяти.
2. Сохранение/загрузка данных в JSON.
3. Поиск и фильтрация записей.
4. Переход на базу данных.
5. Веб-версия на ASP.NET Core.
